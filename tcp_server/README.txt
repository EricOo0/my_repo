#Tcp_server 聊天室服务器程序
#Tcp_client 聊天室客户端程序

客户端用QT编写界面 有注册，登录，聊天室选择，聊天等功能

服务器
	Tcp_server类:  存储服务器建立需要的基本函数和变量  
		       用一个哈希表存储在线的用户fd和对用户名
		       用一个set存放在线用户fd
	epoll+多线程  1个线程负责接收连接请求
		       1个线程负责转发聊天消息
		       1个线程负责http请求——未具体完成
               mysql               连接数据库，验证登录消息和注册写入账号密码功能
	锁 mutex_lock  防止多线程操作出现矛盾
